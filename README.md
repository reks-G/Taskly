# TaskControl - Telegram Mini App

Telegram Mini App для управления задачами с уведомлениями о дедлайнах.

## Возможности

- ✅ Создание задач с приоритетами (низкий, средний, высокий)
- ✅ Установка дедлайнов
- ✅ Редактирование и удаление задач
- ✅ Фильтры: все, активные, сегодня, неделя, просроченные
- ✅ Два режима просмотра: список и доска
- ✅ Календарь задач
- ✅ Уведомления за 15 минут до дедлайна
- ✅ Автоматическая очистка просроченных задач (через 7 дней)
- ✅ Премиум UI с glassmorphism эффектами
- ✅ Полная адаптация под мобильные устройства

## Технологии

- **Backend**: Python + Flask
- **Database**: SQLAlchemy + SQLite
- **Bot**: pyTelegramBotAPI
- **Templates**: Jinja2
- **Timezone**: pytz (Moscow time)
- **Без JavaScript** - все на Python

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Приложение запустится на `http://0.0.0.0:5000`

## Использование

1. Откройте бота в Telegram
2. Нажмите `/start`
3. Нажмите "🚀 Открыть TaskControl"
4. Создавайте и управляйте задачами
5. Получайте уведомления о приближающихся дедлайнах

## Команды бота

- `/start` - Запуск бота и регистрация
- `/tasks` - Показать активные задачи в чате
